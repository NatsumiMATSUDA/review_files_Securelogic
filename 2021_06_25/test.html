HUBER＋SUHNER 新製品 SUCOFLEX® 500シリーズ

HUBER＋SUHNER

航空宇宙/防衛用 RFマイクロ波コネクタ・ケーブルメーカー

1969年に設立されたHUBER+SUHNERは高品質/高信頼性のRFマイクロ波コネクタ、ケーブル、RF/光変換モジュールのメーカーです。防衛・航空宇宙用アプリケーションや地上試験機等、多くの実績も兼ね備えております。

SUCOFLEX®500シリーズ

● SUCOFLEX 500シリーズは最高レベルの満足度を保証する製品です。
● 独自のケーブルとコネクタの設計が、最大70GHzまでで、素晴らしいリターンとインサーションロス特性、らに屈曲、動き、温度、引張応力に対して、最高の位相と振幅の安定性を提供します。
● 特殊な加工、「Rotary swaging」という加工がされた低損失の内部導体と頑丈な構造により、SF500タイプは通常、性能を低下させる事なく10万回以上の屈曲に耐える、長寿命な製品となっています。
●アプリケーション例：
• Bench top testing • RF production testing
• Automated test equipment • Vector network analyzers (VNAs)
• Scalar analyzers • Portable test equipment
• RF module testing • High speed digital testing (HSDT)
